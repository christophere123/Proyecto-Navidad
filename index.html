cartas para santa

Para este proyecto buscamos crear un buzon interactivo como funcion principal, luego complementarlo con musica, luces y el inigualable santa

hicimos uso de 3 sensores, sonido, proximidad y movimiento, con los cual activan la variedad de actuadores, como los zerbos, zumbadores y la pantalla, no pudimos iluminar mediante arduino por que no soportaba tanto voltaje





Base de Datos

Para poder relacionar nuestro proyecto con la materia de construye bases de datos requerimos de realizar nuestra propia base de datos, y para ello haremos uso de un servidor gratuito el cual guardara y manejara los datos

posteriormente programamos los documentos php para crear conexion y de ahi poder ingresar e imprimir los datos requeridos
